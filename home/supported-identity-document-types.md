# Supported Identity document types

The identity system supports a wide range of document types for both `ORGANIZATION` and `USER` across multiple countries. These documents are used to verify the identity of users and the legitimacy of organizations within the platform. Below is a comprehensive list of all supported identity documents, including their scope and description.

### Document Types


| Id document Number | Country (ISO-3) | Applicable To | Description |
| --- | --- | --- | --- |
| PASSPORT | `ALL` | `ORGANIZATION`, `USER` | Standard international passport used for identity verification. |
| RFC | `MEX` | `ORGANIZATION` | Federal Taxpayer Registry used for identifying legal entities in Mexico. |
| CURP | `MEX` | `USER` | Unique 18-digit alphanumeric identifier assigned to both citizens and residents of Mexico |
| RUT | `CHL` | `ORGANIZATION`, `USER` | Unique Taxpayer Role used for both individuals and organizations in Chile. |
| CED_CIU | `COL` | `USER` | National identity card for Colombian citizens. |
| CED_EXT | `COL` | `USER` | Foreign ID card used by non-Colombians residing in Colombia. |
| NIT | `COL` | `ORGANIZATION` | Tax Identification Number used for businesses in Colombia. |
| DNI | `PER`, `ARG` | `USER` | National Identity Document for individuals in Peru and Argentina. |
| CE | `PER` | `USER` | Foreign ID card used by non-Peruvian residents. |
| RUC | `PER` | `ORGANIZATION`, `USER` | Unique taxpayer registry for individuals and organizations in Peru. |
| CUIL | `ARG` | `USER` | Unique Labor Identification Code for employees in Argentina. |
| CUIT | `ARG` | `ORGANIZATION` | Unique Tax Identification Code for legal entities in Argentina. |
